🌐 How Global Save Works

When you click 🌐 Save Global, the site sends your updated data to a Vercel serverless function (/api/save) which commits it to GitHub as public/data/data.js. Vercel detects the commit and rebuilds — every visitor worldwide sees the update in ~30 seconds. No database. No extra services.

✓ Global in ~30 seconds ✓ Full version history on GitHub ✓ Zero extra cost
Note on follower auto-fetch: Instagram and Social Blade block cross-origin browser requests. Follower counts must be entered manually via Edit Mode or via the Bulk Follower Import in Load from JSON. This is a platform restriction, not a bug.
